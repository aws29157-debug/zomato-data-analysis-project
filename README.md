# zomato-data-analysis-project
Zomato Restaurant Data Analysis

This project explores restaurant data from Zomato
to understand restaurant ratings, customer engagement,
cuisine popularity and market concentration.

Tools Used:
Python
Pandas
Matplotlib
Seaborn

Key Insights:
• India dominates the dataset
• Most restaurants rating 3–4 range में हैं
• Higher priced restaurants tend to receive higher ratings
• Customer engagement improves restaurant reputation
